In the console, type:
  gcc -g PL0compiler.c
  ./a.out in.txt (or whatever the input file is named as)

This will generate the file "elf.txt" that contains the code generated by the compiler for the Virtual Machine to run.

To run the code generated by the compiler on the Virtual Machine:

In the console, type:
  gcc -g vm.c
  ./a.out elf.txt
